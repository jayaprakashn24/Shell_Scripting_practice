Real-time Examples:
Variables: Storing user preferences, settings, or other dynamic values in scripts.
$variable: Accessing system paths, user names, or environment variables.
read: Asking users for input in interactive scripts.
Arithmetic Expansion: Performing calculations like adding prices, computing areas, etc.
Command Substitution: Using the result of a system command, like listing files or fetching system information, within another command.

Variables → Values are stored.
$variable → Access the stored value.
read → Accept input from the user.
Arithmetic expansion (()) → Perform mathematical calculations.
Command substitution () → Use the output of a command.
